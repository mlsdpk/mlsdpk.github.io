Obstacles detection is the crucial requirement for the autonomous vechicles. This project implements the 3D object detection pipeline using pure LIDAR sensor only. This is currently an ongoing project. It is based on ROS2 framework and SVL autonomous driving simulator.
{: .text-justify}

You can find more details of the project in my [github repository](https://github.com/mlsdpk/lidar-obstacle-detection).
