A tool for visualizing numerous pathfinding algorithms.

Current implementation of the project involves four search-based planning algorithms: BFS, DFS, DIJKSTRA and A\*, and two sampling-based planning algorithms: RRT and RRT*. The project extensively uses SFML and Modern C++ features such as smart pointers, lamda expressions along with multi-threading concepts.

You can find more details of the project in my [github repository](https://github.com/mlsdpk/path-finding-visualizer). This is an ongoing project and it would be greatly appreciated, if you could contribute to this.
